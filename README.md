# OIBSIP
OIBSIP - Oasis Infobyte Web Development Internship

Overview

This repository contains the projects completed as part of the Web Development and Design Internship at Oasis Infobyte. The projects were developed using HTML, CSS, and JavaScript to strengthen frontend web development skills.

1. Calculator

A simple and responsive calculator that performs basic arithmetic operations.

Features

- Clear and Delete functionality
- Responsive user interface

Technologies Used

- HTML
- CSS
- JavaScript

Features

Technologies Used
